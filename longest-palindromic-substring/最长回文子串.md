# 最长回文子串

2022/05/29

> <https://leetcode.cn/problems/longest-palindromic-substring/>

## 示例

### 示例一

输入：s = "babad"
输出："bab"
解释："aba" 同样是符合题意的答案。

### 示例二

输入：s = "cbbd"
输出："bb"

## 思考过程

这道题完全没有头绪，看看题解尝试理解一下

### 题解思路

```javascript
var longestPalindrome = function (s) {
  const set = new Set(s);
  let head = 0;
  let tail = s.length - 1;
  while (s[tail] !== s[head]) {
    head;
  }
};
```
